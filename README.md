# Wazuh-SIEM-Lab

## Overview

This project aims to demonstrate the design and implementation of a Security Information and Event Management Lab built using Wazuh and deployed on AWS EC2 instances running Ubuntu.

The objective of this lab is to learn how to design and implement security monitoring architecture to collect logs and analyze events to create detection rules to identify suspicious activities.
